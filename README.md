- [ ] **CI**

- Jenkins

- Terraform
- AWS

2. In Jenkins create token and run script 1_Jenkins\/1.2_install_jenkins_plugins.sh It will install plugins:

- GIT
- Maven
- Deploy (pack Java war file to docker container)
